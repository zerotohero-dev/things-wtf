# sed — Practical Recipes

Real-world one-liners and short scripts organized by category.

---

## Text Cleaning

```bash
# Strip ANSI escape codes from terminal output
sed -E 's/\x1B\[[0-9;]*[mK]//g' colored.log

# Normalize Windows CRLF to Unix LF
sed 's/\r//' windows.txt > unix.txt
sed -i 's/\r//' file.txt               # in-place

# Remove trailing whitespace
sed 's/[[:space:]]*$//' file.txt

# Remove leading whitespace
sed 's/^[[:space:]]*//' file.txt

# Remove both leading and trailing whitespace
sed -E 's/^[[:space:]]+|[[:space:]]+$//' file.txt

# Collapse multiple blank lines into one
sed '/./,/^$/!d' file.txt

# Remove all blank lines
sed '/^[[:space:]]*$/d' file.txt

# Remove HTML tags
sed -E 's/<[^>]+>//g' file.html

# Remove XML comments
sed -E 's/<!--[^-]*-->//g; /<!--/,/-->/d' file.xml
```

## Line Operations

```bash
# Print only lines 5-10
sed -n '5,10p' file.txt

# Print every other line (odd lines)
sed -n 'p;n' file.txt

# Print every other line (even lines)
sed -n 'n;p' file.txt

# Add line numbers
sed '=' file.txt | sed 'N;s/\n/\t/'

# Reverse line order (tac)
sed -n '1!G;h;$p' file.txt

# Double-space a file
sed 'G' file.txt

# Remove double-spacing
sed -n 'p;n' file.txt

# Print lines between two patterns (inclusive)
sed -n '/^---$/,/^---$/p' document.md

# Print lines between two patterns (exclusive — exclude the delimiters)
sed -n '/^---$/,/^---$/{/^---$/!p}' document.md

# Delete from a matching line to EOF
sed '/^# DO NOT EDIT BELOW/,$d' config.sh
```

## File Modification

```bash
# Comment out lines matching a pattern
sed '/^LoadModule ssl/s/^/# /' httpd.conf

# Uncomment lines matching a pattern
sed '/^# LoadModule ssl/s/^# //' httpd.conf

# Insert a line after a matching line
sed '/^kind: Deployment/a\  annotations:' manifest.yaml

# Insert a line before a matching line
sed '/^kind: Deployment/i\# Managed by Flux' manifest.yaml

# Prepend a header to a file (in-place)
sed -i '1s/^/# Generated by build.sh\n/' output.sh

# Replace lines 3 to 5 entirely
sed '3,5c\[REDACTED]' file.txt

# Delete a block from /BEGIN/ to /END/
sed '/BEGIN/,/END/d' file.txt

# Remove duplicate consecutive blank lines
sed '/^$/{ N; /^\n$/d }' file.txt
```

## Configuration File Surgery

```bash
# Change a value in a YAML key: value file
sed -i 's/^  replicas: .*/  replicas: 3/' deployment.yaml

# Update a specific Go constant
sed -i 's/const Version = "[^"]*"/const Version = "1.2.3"/' version.go

# Bump semver patch (requires GNU sed /e flag)
sed -E -i 's/(version: "[0-9]+\.[0-9]+\.)([0-9]+)(")/echo "\1$((\2+1))\3"/e' Chart.yaml

# Enable a disabled config line (remove leading #)
sed -i '/^#[[:space:]]*ENABLE_FEATURE/s/^#[[:space:]]*//' config

# Replace a URL across all Go files
find . -name '*.go' -exec sed -i 's|https://old.api.example.com|https://new.api.example.com|g' {} +
```

## Log Processing

```bash
# Extract lines between timestamps
sed -n '/2024-01-15 09:00/,/2024-01-15 10:00/p' app.log

# Remove log timestamps (ISO format)
sed -E 's/^[0-9]{4}-[0-9]{2}-[0-9]{2}T[0-9:\.Z+-]+ //' app.log

# Extract just the message part (everything after third |)
sed -E 's/^([^|]*\|){3}//' structured.log

# Redact passwords from logs
sed -E 's/(password|passwd|secret)[=:][^ ]*/\1=REDACTED/gi' app.log
```

## Code Manipulation

```bash
# Remove single-line C++ comments
sed 's|//.*||' source.cpp

# Remove Python docstrings (simplified, single line)
sed '/""".*"""/d' script.py

# Add indentation to every line
sed 's/^/    /' code.txt

# Remove indentation (4 spaces)
sed 's/^    //' code.txt

# Wrap long lines at 80 chars
sed ':a; s/.\{81,\}/&\n/; ta' file.txt
```

## sed as grep

```bash
# Print only lines matching a pattern (like grep)
sed -n '/ERROR/p' logfile

# Print only non-matching lines (like grep -v)
sed '/ERROR/d' logfile

# Print line number of first match (like grep -n ... | head -1)
sed -n '/PATTERN/=' file.txt | head -1
```
